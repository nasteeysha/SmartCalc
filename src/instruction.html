Smart Calc

О программе

Научный калькулятор с добавленными возможностями построения графиков функций f(x) и задания переменной x в выражениях + Кредитный калькулятор

Установка

Для установки калькулятора введите команду make install

Запуск приложения

После установки приложения оно появится на рабочем столе, где можно его запустить

Удаление

Для удаления калькулятора используйте команду make uninstall
